💡 Challenge:
Create a tuple to store your name (as &str) and your age (as i32).
Then create an array of 3 numbers.
Print both using println!.

⏳ Hint: Use let person = ("Alice", 30); and let numbers = [1, 2, 3];

⏳ Try writing this on your own before you look at the solution!

✅ Show Solution 
fn main() {
    let person = ("Alice", 30);
    let numbers = [1, 2, 3];

    println!("Name: {}, Age: {}", person.0, person.1);
    println!("Numbers: {:?}", numbers);}
