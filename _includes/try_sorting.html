Create a list of students with (name, marks) and:
• sort by marks (ascending)
• then sort by name (alphabetical)
• print both results
Can you do it using sort_by_key() only?

fn main() {
    let mut students = vec![
        ("Riya", 88),
        ("Manu", 95),
        ("Rose", 78),
        ("Anita", 90),
    ];

    // sort by marks
    students.sort_by_key(|s| s.1);
    println!("By marks: {:?}", students);

    // sort by name
    students.sort_by_key(|s| s.0);
    println!("By name: {:?}", students);
}
