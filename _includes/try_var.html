💡 Challenge:
Inside fn main(), declare a variable x with the value 5.
Then, increase x by 1 and print the result using println!.

⏳ Try writing this on your own before you look at the solution!

✅ Show Solution 
fn main() {
    let mut x = 5;
    x = x + 1;
    println!("x is: {}", x);
    }
