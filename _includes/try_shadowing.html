💡 Challenge:
Inside fn main(), declare a variable x with value 10.
Then, use shadowing to double its value.
Finally, print the new value of x.

⏳ Try writing this on your own before you look at the solution!

✅ Show Solution 
fn main() {
    let x = 10;
    let x = x * 2;
    println!("x is: {}", x);
    }
